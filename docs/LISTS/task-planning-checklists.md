# TASK PLANNING CHECKLISTS AND WORKPLANS

Use this page as a template for turning ambiguous work into a checklist that also includes numbered steps for execution.

## Checklist for kickoff

- [ ] Intake notes
  - [ ] Who is the audience?
  - [ ] What is the desired outcome?
- [ ] Constraints
  - [ ] Time window
  - [ ] Allowed formats
  - [ ] Done definition

## Numbered workplan with sub-bullets

1. Draft the outline
   - Create headings for sections
   - Decide where lists appear
2. Fill in examples
   - Add bulleted examples
   - Add numbered examples
   - Add checklist examples
3. Review rendering
   - Confirm indentation depth
   - Ensure checklist markers stay aligned

## Acceptance checklist (use it as a final gate)

- [ ] Every example uses one of: bullets, numbers, checkboxes
- [ ] At least one list demonstrates deep indentation
- [ ] At least one list demonstrates mixed nesting
- [ ] No list item contains contradictory instruction
